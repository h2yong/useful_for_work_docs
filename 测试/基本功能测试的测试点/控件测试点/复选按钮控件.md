# 复选按钮控件

| 编号 | 输入/动作      | 期望的输出/相应                                                                                           |
| :--- | :------------- | :-------------------------------------------------------------------------------------------------------- |
| 1    | 显示风格       | 1．选择框统一居左显示，名称描述统一居右显示                                                               |
| 2    | 使用风格       | 1．如果能通过一个CHECKBOX框，即可实现两个功能，那么不允许使用两个CHECKBOX框完成一个功能【如：显示已作废】 |
| 3    | 选择一个复选框 | 1．选择有效                                                                                               |
| 4    | 选择多个复选框 | 1．系统允许此项操作                                                                                       |
