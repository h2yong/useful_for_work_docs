# 列表框控件
| 编号 | 输入/动作                                | 期望的输出/相应                                  |
| :--- | :--------------------------------------- | :----------------------------------------------- |
| 1    | 显示在列表框的内容                       | 内容根据需求说明确定                             |
| 2    | 列表框显示的内容超过列表框显示的区域     | 列表框要及时出现滚动条。【包括垂直、横向滚动条】 |
| 3    | 使用键盘选择列表框中的数据               | 1．系统要提供通过键盘进行选择的功能              |
| 4    | 使用Shift键，选择列表框的多条数据        | 1．系统要提供此项选择功能                        |
| 5    | 使用Ctrl键，选择性的选择列表框的多条数据 | 1．系统要提供此项选择功能                        |
